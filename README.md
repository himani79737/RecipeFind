# 🍽️ Recipe Finder App

A simple and interactive web application that allows users to search for recipes and view details instantly using a public API.

## 🚀 Features

- 🔍 Search recipes by name
- 📡 Fetch data from TheMealDB API
- ⚡ Fast and dynamic UI updates
- 💾 Saves last searched recipe using localStorage
- ❌ Displays message when no results found

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript (ES6)
- REST API (TheMealDB)

## 📸 How It Works

1. User enters a recipe name
2. App sends a request to the API
3. Recipes are displayed dynamically on the screen
4. Last search is saved in localStorage

## 📂 API Used

https://www.themealdb.com/api.php
